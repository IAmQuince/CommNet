# Launcher Specification


[SEC:CMN-APP-SPEC-002::1]
The shell provides BAT launchers for configuration, portal, shutdown, diagnostics, and support export.

[REQ:CMN-REQ-LAUNCH-001]
Launchers shall use relative paths and Python discovery rather than hard-coded installation paths.
