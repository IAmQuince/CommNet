# Local Shell Specification


[SEC:CMN-APP-SPEC-001::1]
The local shell shall launch from a flash-drive extracted folder without requiring pip packages.

[REQ:CMN-REQ-SHELL-001]
CommNet shall launch a local server with Python standard-library components in the shell package.

[REQ:CMN-REQ-SHELL-002]
CommNet shall bind to `127.0.0.1` by default for this local demonstrator.
