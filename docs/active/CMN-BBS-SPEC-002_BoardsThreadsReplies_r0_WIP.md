# BBS Boards, Threads, and Replies Specification

[SEC:CMN-BBS-SPEC-002::PURPOSE]
The BBS shall become a usable local message board rather than a placeholder page. It supports board navigation, thread lists, thread details, and replies.

[SEC:CMN-BBS-SPEC-002::BOARDS]
The seeded board structure includes Announcements, General, Help, RetroWeb, and Emergency / Outage. The Announcements board contains an initial welcome thread that demonstrates the interface.

[SEC:CMN-BBS-SPEC-002::THREADS]
Users may create a thread with title and body. Thread lists show board, author, reply count, and last activity. Thread detail pages show the original post plus replies.

[SEC:CMN-BBS-SPEC-002::FUTURE-MODERATION]
Pinning, locking, deletion, deeper moderation, paging controls, and search are future extensions. The current run focuses on navigable local structure and smoke-tested persistence.
