# Community Web Portal Entry Specification

| Revision | Date | Status | Summary |
|---|---|---|---|

## 1. Purpose [SEC:CMN-UI-SPEC-004::1]

The community web portal is the user-facing entry point for browsing local resources, services, community pages, directories, and approved content.

## 2. Separation from Admin [SEC:CMN-UI-SPEC-004::2]

The portal shall not be the same as the admin configuration platform. A normal user may browse a portal without receiving permission to alter privacy, roles, gateway exposure, or system services.

## 3. Visibility Statement [SEC:CMN-UI-SPEC-004::3]

The portal should clearly show whether the current user/node is visible in the community directory and what that means.
