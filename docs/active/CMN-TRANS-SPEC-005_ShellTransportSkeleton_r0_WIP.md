# Shell Transport Skeleton


[SEC:CMN-TRANS-SPEC-005::1]
The shell package shall include the transport adapter interfaces and declared adapters needed to prevent architecture drift.

Functional in this package:
- Local loopback adapter.

Declared for later packages:
- LAN/Wi-Fi HTTP.
- Meshtastic Serial, TCP, BLE, MQTT.
- Reticulum/RNS.
- LXMF.
- Bluetooth BLE.
- Generic serial.
- Storage node.
- Removable media.
- Phone cache.
- Drone/data mule.
